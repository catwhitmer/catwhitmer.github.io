To be honest, Javascript made me realize how much I enjoy Ruby and Rails. Some of the concepts were very hard to grasp, such as "this". It was also challenging to keep code organized since JS allows you so much free reign. Rails is very streamlined which helps to understand your code's purpose.

For my JS project I created a Hogwarts Student Roster application that allows a teacher to view, edit, and delete students from the roster. The backend API was developed with Ruby Active Record, and the frontend was created using JavaScript. Both communicate through AJAX rendering in JSON.

Creating the backend was a breeze. One thing that was interesting was implementing a serializer. The serializer added a fastJson gem and I really had no clue what it was and what it would do to my code rendering. It turns out that it not only renders Jsaon quickly, but it also lets you customize the look of your code you want rendered by being able to set I.D.'s, attributes, and many other relationships. I loved playing around with this feature.

The frontend was not as easy. This is where I struggled the most. I got stuck many times. One thing that really helped was the debugger. It makes it easier to look at what your code is doing and how you can manipulate it. It was actually quite fun. I also didn’t realize that you can set up the frontend many different ways, which made me miss Ruby again for being predictable.

JS was definitely challenging but I can also see its vast potential. I feel like I still have a lot to learn but that's the fun part. This project helped me understand JS a lot more and I can’t wait to keep on learning.
