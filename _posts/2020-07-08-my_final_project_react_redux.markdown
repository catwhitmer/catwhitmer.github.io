Wow. I can't believe I just completed my last project. It feels like only yesterday that we were learning about functions and variables. It is mind boggling to think how much we learned in a few short months.
 
For my final project, I really wanted to create an application that I could use in my personal life. Having three dogs, it's hard to keep track of shots, medications, baths, etc. Pet Care is an app that lets you create a list of pets and add todos to your individual pet. Each pet has its own customized page and it's easy to navigate between pages.

The backend API was made with Rails using two models: Pet and To-Do. Serializers were also used to display the information that was needed and to take advantage of the nested routes. The only extra gem that I used was pry, to check if the delete and add actions were coming through correctly.

The frontend was created with React/Redux and was a bit challenging but fun to build with. Using the react and redux dev tools is an absolute must. I could not have done this project without them. They really help you see the props in each component and the current  state. Routes props can be especially useful in redirecting routes and in getting the information you need in rendering a certain property.

This project, although challenging at times, was a joy to build. I learned so much from this project alone and it really helped me think more like a programmer.



What I learned:
* React/Redux Dev Tools are awesome!
* debugger and pry are lifesavers
* Really learn the terms props and state
